DummyPubsub()

Dummy Pubsub used for instrumenting code without connecting. - does not extend Pubsub, this is a standalone object - this object is created instead of actual Pubsub when user code is initialized via Code.run(). - when user code is executed via Code.run(), it has an IPC channel to the master process. in that case, pubsub.publish is done by the master process, and the child process (this process) sends the message to the master process.
